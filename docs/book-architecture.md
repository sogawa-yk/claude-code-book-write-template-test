# 書籍構成・章間依存関係

## 章構成

| 章 | タイトル | ページ | 性質 |
|----|---------|--------|------|
| 第1章 | WASMの正体 | 8p | 概念・背景 |
| 第2章 | 最小実装から理解する | 10p | ハンズオン |
| 第3章 | JavaScriptとの連携 | 12p | ハンズオン |

| 第5章 | 本番環境への道 | 10p | 運用・デプロイ |

## 章間依存関係

```
第1章（概念）
  └─→ 第2章（基礎実装）
        └─→ 第3章（JS連携）
              └─→ 第4章（応用）
                    └─→ 第5章（本番）
```

本書は完全な直列構成である。各章が前章の知識とサンプルアプリの状態に依存するため、順序通りに読むことを前提とする。
