# 書籍企画書 (Book Plan)

## 書籍概要

### タイトル
**Python入門ハンズオン** ― サンプルコードで学ぶプログラミングの基礎

### 形態
- 判型: A5
- 想定ページ数: 20ページ
- 配布形態: 技術書典 / 電子配布

### コンセプト
- **手を動かして学ぶ**: すべての概念をサンプルコードの実行を通じて体験する
- **最短距離**: 必要最低限の知識に絞り、20ページで基礎を習得する
- **実用重視**: 学んだ直後に使える実践的なコード例を提供する

### ビジョン
プログラミング未経験者や他言語経験者がPythonの基礎を短時間で習得できる入門書。理論の説明は最小限にとどめ、サンプルコードを実際に動かしながら言語の基本構文とデータ構造を身につける。20ページというコンパクトな分量で、読者が「Pythonで簡単なプログラムを書ける」状態に到達することを目指す。

## 対象読者

### プライマリーペルソナ
- プログラミング初学者、または他言語からPythonに入門したい開発者
- 技術レベル: 初級（PCの基本操作ができる）
- 前提知識: テキストエディタの使い方、コマンドラインの基本操作
- 課題: Pythonの基本構文を短時間で把握し、自分でコードを書けるようになりたい

### 前提知識
読者が本書を読む前に理解していることを前提とする:
- PCの基本操作（ファイル作成・保存）
- テキストエディタの使用経験
- コマンドライン（ターミナル）で文字を入力・実行できる

### 前提としない知識
本書で説明する:
- Pythonのインストールと実行環境の準備
- プログラミングの基本概念（変数、条件分岐、ループ等）

## 到達目標

読了後、読者が以下をできるようになること:

1. Pythonの実行環境をセットアップし、スクリプトを実行できる
2. 変数、データ型、演算子を使った基本的な処理を書ける
3. 条件分岐（if文）とループ（for/while）で制御フローを組める
4. リスト・辞書などのデータ構造を活用できる
5. 関数を定義して処理をまとめられる

## 全体構成

### 章構成

| 章 | タイトル | ページ（目安） | 性質 |
|----|---------|--------------|------|
| 第1章 | Python環境のセットアップ | 3ページ | 導入・環境構築 |
| 第2章 | 変数とデータ型 | 4ページ | 基礎知識 |
| 第3章 | 制御フロー | 5ページ | 基礎知識 |
| 第4章 | データ構造 | 4ページ | 基礎知識 |
| 第5章 | 関数 | 4ページ | 基礎知識・実践 |

## 技術環境

- **主要な技術スタック**: Python 3.12+
- **コード例の言語**: Python

## 執筆方針

- **コードファースト**: 概念の説明より先にコード例を提示し、動かしてから解説する
- **段階的な難易度**: 各章が前章の知識を前提とし、無理なくステップアップする
- **コンパクト**: 1つのトピックにつき1〜2個の厳選されたサンプルコードで解説する

## スコープ外

明示的にスコープ外とする項目:
- オブジェクト指向プログラミング（クラス、継承）
- 外部ライブラリ（pip、パッケージ管理）
- ファイル入出力、例外処理
- Web開発、データ分析等の応用分野

## 成功指標

- 読者が20ページを通読後、自力でFizzBuzz程度のプログラムを書ける
- 各章のサンプルコードをコピー&実行して全て動作する
