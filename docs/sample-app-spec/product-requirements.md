# WASM Image Filter ― 全体仕様

## 概要

画像フィルタWebアプリケーション。Rust製WASMモジュールで画像処理を行い、JavaScriptのUIから操作する。

## 機能一覧

| 機能 | 説明 | 導入章 |
|------|------|--------|
| 画像読み込み | ローカルファイルからの画像読み込み（File API） | 第2章 |
| グレースケール変換 | RGB→輝度値変換 | 第2章 |
| セピアフィルタ | 暖色系色調変換 | 第3章 |
| ぼかしフィルタ | ガウシアンブラー（3x3カーネル） | 第3章 |
| プレビュー表示 | Canvas上でのリアルタイムプレビュー | 第3章 |
| バッチ処理 | 複数画像の一括フィルタ適用 | 第4章 |
| 性能計測 | WASM vs JS実装の処理時間比較表示 | 第4章 |
| バンドル最適化 | wasm-opt、Tree Shaking適用 | 第5章 |

## 非機能要件

- **対応ブラウザ**: Chrome、Firefox、Safari（最新版）
- **対応画像形式**: PNG、JPEG
- **最大画像サイズ**: 4096x4096px
- **WASMバイナリサイズ目標**: 最適化後100KB以下

## 制約

- サーバーサイド処理は不要（全てクライアントサイドで完結）
- 第5章のKubernetesデプロイではWASI対応のCLI版を別途作成
