# サンプルアプリ構想: WASM Image Filter

## 概要

ブラウザ上で画像にフィルタを適用するWebアプリケーション。画像処理ロジックをRustでWASMモジュールとして実装し、JavaScriptのUIから呼び出す。

## 目的

- WASMの基本的なビルド・実行フローを体験する
- JavaScript-WASM間のデータ受け渡し（特に画像データ＝バイト配列）を理解する
- CPU集約的タスクにおけるWASMの性能優位性を実感する
- 本番環境（Kubernetes）へのデプロイを体験する

## 機能

1. **グレースケール変換**: ピクセルのRGB値を輝度値に変換（第2章）
2. **セピアフィルタ**: 暖色系の色調に変換（第3章）
3. **ぼかしフィルタ**: ガウシアンブラー（第3章）
4. **バッチ処理**: 複数画像の一括フィルタ適用（第4章）

## 章ごとの進化

| 章 | 追加機能 | 学習ポイント |
|----|---------|------------|
| 第2章 | グレースケール変換 | wasm-pack、基本的なWASMビルド |
| 第3章 | セピア、ぼかし | メモリ共有、型変換、エラーハンドリング |
| 第4章 | バッチ処理、性能比較 | パフォーマンス計測、適用判断 |
| 第5章 | 最適化、K8sデプロイ | バンドルサイズ、containerd+runwasi |
