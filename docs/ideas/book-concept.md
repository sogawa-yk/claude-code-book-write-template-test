## 📚 本の全体構成（5 Parts）

### **Part 1: WASMの正体**

- 概念説明: バイナリフォーマット vs 言語
- なぜWASMか: JavaScript の限界
- **Bridge**: Docker/Kubernetes時代での重要性

### **Part 2: 最小実装から理解する**

- **Rust + wasm-pack** による Hello WASM（メイン例）
- **言語比較**: Python (Pyodide), Go (TinyGo), C/C++ を並列学習
- ツールチェーン理解: wasm-pack, wasm-bindgen, wasmtime

### **Part 3: JavaScriptとの連携**

- WASM関数の呼び出し、型変換、エラーハンドリング
- メモリ共有（ArrayBuffer）の仕組み
- **実例**: 画像フィルタ処理（Rust ↔ JavaScript）

### **Part 4: 実用アプリケーション**

- **4-1**: CPU集約的タスク（素数判定、フィボナッチ）
- **4-2**: バイナリ処理（圧縮・解凍）
- **4-3**: 科学計算（NumPy互換）
- **4-4**: リアルタイムメディア処理

### **Part 5: 本番環境への道**

- バンドルサイズ最適化
- ブラウザ互換性対応
- **Kubernetes でのWASM実行**（簡易 Pod デプロイ例）
- パフォーマンス分析
