## 節の構成

### 2.1 変数と代入

**概要**: 変数の概念と代入の仕組みを説明する。変数名の命名規則、再代入、動的型付けについて扱う。

**図表**:
- 図2.1: 変数と値の関係（名前→オブジェクトの参照モデル）

**キーポイント**:
- 変数は値に名前を付ける仕組みである
- Pythonは動的型付け言語であり、型宣言が不要である
- 変数名にはスネークケース（`my_variable`）を使用する

### 2.2 基本データ型

**概要**: Python の基本的なデータ型（int, float, str, bool）を紹介する。`type()`関数で型を確認する方法を示す。

**図表**:
- 表2.1: 基本データ型の一覧と特徴

**キーポイント**:
- `int`は整数、`float`は小数、`str`は文字列、`bool`は真偽値
- `type()`関数で値の型を確認できる
- 型変換（`int()`, `str()`等）で型を変換できる

### 2.3 演算子

**概要**: 算術演算子、比較演算子、論理演算子を紹介する。演算子の優先順位と文字列の結合についても扱う。

**図表**:
- 表2.2: 演算子の種類と動作一覧

**キーポイント**:
- 算術演算子: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- 比較演算子: `==`, `!=`, `<`, `>`, `<=`, `>=`
- 論理演算子: `and`, `or`, `not`
- 文字列は`+`で結合、`*`で繰り返しができる
