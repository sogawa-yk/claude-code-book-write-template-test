# 第2章 最小実装から理解する ― Rust + wasm-pack ― 仕様書

## 章の概要

**章の目的**: Rust + wasm-packでWASMモジュールをビルド・実行する一連の流れを体験し、ツールチェーンの役割を理解する。Go（TinyGo）やPython（Pyodide）との比較も行う。

**目標ページ数**: 12p

**所属する部**: 第II部 実装入門

- コマンドラインツールの基本操作
