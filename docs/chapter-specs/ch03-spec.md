# 第3章 制御フロー ― 仕様書

## 章の概要

**章の目的**: 条件分岐とループを使ってプログラムの実行順序を制御できるようになること。

**目標ページ数**: 5p

## 前提知識

この章を読むために必要な前提:
- 第2章の内容（変数、データ型、比較演算子、論理演算子、ブール型の理解）

## 節の構成

### 3.1 条件分岐（if文）

**概要**: `if`/`elif`/`else`文による条件分岐を説明する。インデントによるブロック構造、ネストした条件分岐も扱う。

**図表**:
- 図3.1: 条件分岐の実行フロー（if/elif/else）

**キーポイント**:
- `if`文は条件式が`True`の場合にブロック内を実行する
- `elif`で複数条件を判定、`else`でどの条件にも合致しない場合を処理
- Pythonではインデント（4スペース）がブロックを定義する

### 3.2 forループ

**概要**: `for`文によるイテラブルの反復処理を説明する。`range()`関数、`enumerate()`の使い方を紹介する。

**図表**:
- 図3.2: forループの実行フロー

**キーポイント**:
- `for 変数 in イテラブル:`の構文で要素を順に処理する
- `range()`で連続する整数のシーケンスを生成できる
- `enumerate()`でインデックスと要素を同時に取得できる

### 3.3 whileループ

**概要**: `while`文による条件ベースの繰り返しを説明する。無限ループと`break`/`continue`による制御も扱う。

**図表**:
- 図3.3: whileループの実行フローとbreak/continueの動作

**キーポイント**:
- `while`文は条件式が`True`の間ブロックを繰り返す
- `break`でループを中断、`continue`で次のイテレーションにスキップ
- 無限ループには必ず終了条件を設ける

### 3.4 制御フローの組み合わせ

**概要**: 条件分岐とループを組み合わせた実践的な例を示す。FizzBuzzを題材に、学んだ知識を統合する。

**図表**:
- 図3.4: FizzBuzzのフローチャート

**キーポイント**:
- ループ内で条件分岐を使うパターンは頻出する
- FizzBuzzは制御フローの理解度を測る代表的な問題である

## 章末の理解度チェック問題

### 問題の方針
- 問題数: 4問
- 問題の種類:
  - 概念の確認、判断問題、設計問題

### 問題案
1. `for`ループと`while`ループの使い分けの基準を説明せよ（種類: 概念の確認）
2. 与えられたコードの実行結果を予測せよ（種類: 判断問題）
3. `break`と`continue`の違いを説明せよ（種類: 概念の確認）
4. 1から100までの偶数の合計を求めるコードを書け（種類: 設計問題）
