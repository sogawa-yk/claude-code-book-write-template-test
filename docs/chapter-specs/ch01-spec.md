# 第1章 Python環境のセットアップ ― 仕様書

## 章の概要

**章の目的**: Pythonの実行環境を構築し、最初のコードを実行できるようになること。

**目標ページ数**: 3p

## 前提知識

この章を読むために必要な前提:
- PCの基本操作（ファイル作成・保存）
- コマンドライン（ターミナル）でコマンドを入力できる

## 節の構成

### 1.1 Pythonのインストール

**概要**: OS別のPythonインストール手順を説明する。公式サイトからのダウンロードとインストール後のバージョン確認までを扱う。

**図表**:
- 表1.1: OS別インストール方法の一覧

**キーポイント**:
- Python 3.12以上を使用する
- `python --version`でインストール確認ができる

### 1.2 対話モードとスクリプト実行

**概要**: Pythonの二つの実行方法（対話モードとスクリプト実行）を紹介する。対話モードで即座に結果を確認する方法と、`.py`ファイルにコードを書いて実行する方法を説明する。

**図表**:
- 図1.1: 対話モードとスクリプト実行の使い分けフロー

**キーポイント**:
- 対話モード（REPL）は試行錯誤に適している
- スクリプト実行はまとまった処理に適している
- `python`コマンドで対話モード、`python ファイル名.py`でスクリプト実行

### 1.3 最初のプログラム

**概要**: `print()`関数を使って「Hello, World!」を出力する。対話モードとスクリプトの両方で実行し、動作を確認する。

**図表**:
- 図1.2: プログラムの実行フロー（コード→インタプリタ→出力）

**キーポイント**:
- `print()`は値を画面に出力する関数である
- プログラムは上から順に実行される
